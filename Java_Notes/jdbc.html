JDBC: Java Database Connectivity

The foundational API for database interaction in Java. Learn to build the bridge between your code and data server.

01. What is JDBC?

JDBC is a standard Java API that allows applications to connect to a wide variety of relational databases using SQL.

Core Philosophy

"Write once, connect to any database server via standard SQL."

  • Connect to DB
  • Execute Queries
  • Process Data

02. Architecture Layers

JDBC API

The standard interface provided by Java (java.sql). It's what you write in your code.

JDBC Driver

Database-specific logic provided by vendors like MySQL. It translates API calls to DB-specific commands.

03. Core Classes & Interfaces

Component Responsibility
DriverManager Creates connections based on DB URLs.
Connection Maintains a session with the database.
Statement Executes simple SQL queries.
PreparedStatement Handles precompiled, secure queries.
ResultSet Acts as a cursor for query results.

04. Step-by-Step Workflow

Full JDBC Cycle 
// 1. Load Driver
Class.forName("com.mysql.cj.jdbc.Driver");

// 2. Connect
Connection con = DriverManager.getConnection(url, user, pass);

// 3. Execute
Statement stmt = con.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM users");

// 4. Process & Close
while(rs.next()) { ... }
con.close();
