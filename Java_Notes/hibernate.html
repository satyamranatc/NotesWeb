01. What is Hibernate?

Hibernate is an ORM (Object Relational Mapping) framework for Java. It bridges the gap between object-oriented models and relational databases.

The Logic

"Work with Java Objects. Hibernate handles the SQL."

Traditional JDBC

String sql = "INSERT INTO users VALUES (1, 'John')";
stmt.executeUpdate(sql);

Modern Hibernate

User user = new User(1, "John");
session.save(user);

03. Architecture Components

SessionFactory

The factory for creating sessions. Immutable and heavy—usually one per application context.

Session

Represents a single conversation with the database. Lightweight and used for transient units of work.

04. Core Operations

User.java & CRUD 
User user = new User(1, "John");

session.save(user);    // CREATE
session.get(User.class, 1); // READ
session.update(user);  // UPDATE
session.delete(user);  // DELETE
